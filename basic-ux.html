Basic UX

04.02.01. Design sprint

The design sprint is a 5 day process of answering critical business questions in order to produce a fully functioning prototype. Using these research and design methods my group and I designed an improved interface of Fronter.

04.03.01. Web prototype

Based on the final prototype of the design sprint I created a mobile first web prototype. I chose to use html, css and javascript to build this site. I further developed my skills in back end design from this project.

04.04.01. Design sprint: Native app

In this project we had to design a fully functioning learning based XD prototype with no restrictions. We used knowledge from the previous design sprint to create this prototype. My role in this was to design the news page of the app.

04.04.02. Theme documentation

The Theme Documentation includes the whole process of the UX theme. The documentation shows the knowledge I have learnt and showcases different user tests and results.

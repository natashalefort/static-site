02.01.02 XD prototype

In the Bike Like a Dane XD prototype we used gestalt principles to produce a prototype for our microsite. Through the use of these principles we were able to design a site that relays good sensory information.

02.01.03: Story without words

In this project we learnt about the basics of storytelling. Our group mainly relied on the power of sound and film angles to dramatize the plot. From this project I learnt basic film language and how to edit and shoot a small movie.

02.01.05 Treatment

The task was to write a treatment for a video that is to be implemented on the Bike Like a Dane microsite. From this task I gained more knowledge regarding how to write a treatment and how to correctly present it in a professional manner.

02.01.07 Storyboard

In this task we explored how to create a storyboard for our short movie. I got insight on what elements to include in a storyboard, such as duration, angle and audio.

02.01.08 Documentation

In this theme I learnt how to create a documentation of the creative work. This allowed me to create a documentation of our Bike Like a Dane project. The documentation relays the theme/idea to aid the viewer in understanding the concept.

02.01.09 Links to micro site

The microsite is created with html, css and javascript. It is a responsive microsite relaying information about the bike culture in Denmark. My role was to design the interface of the microsite.
